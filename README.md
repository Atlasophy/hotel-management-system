# Hotel Management System

A hotel management system built with PHP and MySQL.

## Features

- Add Guests
- View Guest List
- MySQL Database Integration
- PHP Backend
- Responsive User Interface

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- Git
- GitHub

## Installation

1. Clone the repository
2. Install MySQL
3. Create the hotel_management database
4. Create the guests table
5. Run:

bash php -S localhost:8000 

6. Open:

http://localhost:8000
